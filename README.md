# Assetly - DeFi Portfolio Management & Analysis

Assetly is an intelligent DeFi portfolio management platform that helps users analyze their crypto holdings, discover yield opportunities, and make data-driven investment decisions.

## Video demo

[https://assetly-ai-agent.vercel.app/](https://www.youtube.com/watch?v=CrnL4Ik1GKo)

## Demo link

https://assetly-ai-agent.vercel.app/


## Features

### 1. Portfolio Inspection
- Connect your wallet to view all your assets
- Automatic token detection and balance tracking
- Separate views for native assets and tokens
- Real-time portfolio valuation

### 2. Market Analysis
- Real-time market sentiment analysis
- Token-specific news and events tracking
- Price movement analysis
- Impact assessment of major market events
- Source-weighted news credibility system

### 3. DeFi Yield Analysis
- Discover the best yield opportunities across protocols
- Real-time APY tracking
- TVL (Total Value Locked) monitoring
- Risk assessment for different protocols
- Protocol performance metrics

### 4. Smart Recommendations
- Personalized investment recommendations
- Action items based on market conditions
- Risk analysis and mitigation strategies
- Yield optimization suggestions
- Buy/Sell signals based on market sentiment

### 5. DeFi Integration
- Direct integration with AAVE for lending
- Uniswap integration for token swaps
- One-click deposit functionality
- Real-time transaction status tracking
- Cross-chain support (Ethereum Sepolia & Arbitrum Sepolia)

## Technical Features
- Web3 wallet integration
- Real-time data feeds from multiple sources
- AI-powered market analysis
- Source-credibility weighted news analysis
- Dynamic threshold system for trading signals
- Terminal-style logging system

## Supported Networks
- Ethereum Sepolia
- Arbitrum Sepolia

## Security Features
- Source credibility verification
- Risk assessment for all protocols from DefiLlama
- Minimum TVL requirements
- Transaction confirmation system
- Real-time error handling and alerts

## Getting Started
